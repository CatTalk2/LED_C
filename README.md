# LED_C
C语言模拟LED交通灯

###1.LED模拟交通灯（51，pic16f877A不限）

12只LED分东西向和南北向两组，各组指示灯均有相同的2只红色、2只黄色与2只绿色的LED. 要求模拟十字路口交通信号灯的切换过程与显示效果，绿灯亮，黄灯闪烁，红灯亮，交替进行。电路原理图（51）参考如下。
